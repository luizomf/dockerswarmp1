# Inbox (Ideias e Pendencias)

Objetivo: um lugar simples para voce anotar ideias sem transformar isso em um
`PLAN.md`. Aqui e "backlog/inbox".

Regra do projeto:
- A gente **so implementa** quando um item daqui for promovido para `PLAN.md`.
- `PLAN.md` sempre descreve **apenas** o que esta sendo feito agora e deve ser
  limpo ao finalizar.

Como adicionar um item:
- escreva uma linha curta
- opcional: contexto + objetivo + risco
- se for algo sensivel (seguranca/infra), marque com `security`/`infra`

Template recomendado:

```text
- [ ] <titulo curto> (tags: infra|app|video|docs|ci|security)
  Contexto:
  Objetivo:
  Notas:
```
